# hashcat-password-simulator 
This project is a Python-based simulation of password cracking techniques inspired by Hashcat, developed as part of NIS IA#2 (TY B.Tech, Jan–Apr 2026) by Gauri Deshmukh, Girish Chaudhari, and Gurkaran Singh Ahuja. It demonstrates how weak passwords can be identified and cracked using methods like dictionary attack, brute-force, mask attack, and rule-based attack, while also providing features such as hash generation (MD5, SHA1, SHA256, SHA512), password strength analysis, salted vs unsalted hashing comparison, and JSON report generation through an interactive CLI. The project highlights key cybersecurity concepts, emphasizing the importance of strong, complex passwords and secure hashing practices.
